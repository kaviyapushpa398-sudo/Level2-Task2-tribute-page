# 🌟 Tribute Page for Sundar Pichai

A simple and elegant **Tribute Page dedicated to Sundar Pichai**, the CEO of Google and Alphabet.
This project highlights his inspiring journey, achievements, and contributions to the world of technology through a well structured and visually appealing web page.

## ✨ Features

* 👤 Introduction section about Sundar Pichai
* 📖 Biography and inspirational journey
* 🏆 Highlights of major achievements
* 🎨 Clean and attractive webpage design
* 📱 Responsive layout for better viewing on different devices

## 🛠 Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript

## 🎯 Purpose of the Project

The goal of this project is to practice **web page structuring, layout design, and styling** while creating a meaningful tribute page for an inspiring technology leader.
